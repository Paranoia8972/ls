# `ls` exaples

The Unix command `ls`recreated in diffrent programming languages


## Language

- [x] [**Rust**](#rust)

- [ ] [**Go**](#go)




## Usage

### Rust

```bash
cd rust

# Building
cargo build --release

# Running
./target/release/rls
```
**Optional:**
Installing
```bash
sudo cp target/release/rls /usr/local/bin/
```
